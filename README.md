# LUXE - Luxury E-commerce App

LUXE is a premium, modern, and fully responsive E-commerce application built with Flutter. It offers a seamless shopping experience for luxury items, featuring a sophisticated UI and robust state management.

## 🚀 Key Features

| Feature | Description |
| :--- | :--- |
| **Onboarding** | Smooth welcome screens introducing the app's premium values. |
| **Auth Flow** | Complete Login & Signup with simulated secure authentication. |
| **Discovery** | Advanced Search, Category browsing, and Price/Rating sorting. |
| **Cart & Bag** | Real-time calculations, Promo codes, and Item management. |
| **UX/UI** | Dark Mode, Hero Animations, and Material 3 Design. |
| **Responsive** | Optimized for Mobile, Tablets, and Web browsers. |

## 🛠️ Tech Stack

| Tool | Usage |
| :--- | :--- |
| **Flutter** | High-performance cross-platform UI framework. |
| **Provider** | State management for reactive data handling. |
| **Material 3** | Latest design system for premium look and feel. |
| **Responsive UI** | Custom utility for multi-device layout support. |

## 🛠️ Installation

1. **Clone & Setup**:
   ```bash
   git clone https://github.com/yourusername/luxe_app.git
   cd luxe_app
   flutter pub get
   ```
2. **Run Application**:
   ```bash
   flutter run
   ```

## 📂 Project Structure

- `lib/core`: Themes, Routes, and Responsive utilities.
- `lib/providers`: State management logic.
- `lib/screens`: All UI screens.
- `lib/widgets`: Reusable UI components.
